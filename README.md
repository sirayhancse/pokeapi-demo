# PokeAPI Demo

PokeAPI is a demo project for using the example of PokeAPI.

## API Docs
https://pokeapi.co/

## Installation

Use the yarn for installation packages

```bash
$ yarn install
```

## Run

```bash
$ yarn dev
```

## Build
```bash
$ yarn build
```
